# react-vejr-app

Den nærværende repo er Marie-Pierre Lessards opgave, som er afleveret graduelt på adressen:
[react-vejr-app på GitHub](https://github.com/male9-techcollege/react-vejr-app).

Til opgaven skal eleverne bruge [den gratis vejr-API fra Open-Meteo.com](https://open-meteo.com). Open-Meteos licens er [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/). Som konsekvens er den nærværende vejr-app beskyttet af den vedlagte MIT-licens. Der bliver ingen ændring til Open-Meteos API, da opgaven udelukkende er en frontend-opgave.
