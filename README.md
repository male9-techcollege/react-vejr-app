# react-vejr-app

Denne repo indeholder Marie-Pierre Lessards opgave. Opgaven er afleveret graduelt på adressen:
[https://github.com/male9-techcollege/react-vejr-app](https://github.com/male9-techcollege/react-vejr-app).

Til opgaven skal eleverne bruge [den gratis vejr-API fra Open-Meteo.com](https://open-meteo.com). Open-Meteos licens er [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/). Som konsekvens er den nærværende vejr-app beskyttet af den vedlagte MIT-licens. Der bliver ingen ændring til Open-Meteos API, da opgaven udelukkende er en frontend-opgave. 

Open-Meteo bruger data fra Danmarks Meteorologiske Institut (DMI). [DMI's data må ikke bruges til kommercielle formål.](https://open-meteo.com/en/docs) 

TechColleges breddegrad og længdegrad blev fundet i OpenStreetMaps database. Disse data er under en licens, som hedder ["Database Contents License (DbCL) v1.0"](https://opendatacommons.org/licenses/dbcl/1-0).
